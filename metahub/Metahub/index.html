    listAsset

    function listAsset(struct Assets.Asset asset, struct Listings.Params params, uint32 maxLockPeriod, bool immediatePayout) external returns (uint256 listingId, uint256 listingGroupId)

    Performs new asset listing. Emits an {AssetListed} event.

    Name Type Description
    asset struct Assets.Asset Asset to be listed.
    params struct Listings.Params Listing strategy parameters.
    maxLockPeriod uint32 The maximum amount of time the original asset owner can wait before getting the asset back.
    immediatePayout bool Indicates whether the rental fee must be transferred to the lister on every renting. If FALSE, the rental fees get accumulated until withdrawn manually.
    Name Type Description
    listingId uint256 New listing ID.
    listingGroupId uint256 Listing group ID.
